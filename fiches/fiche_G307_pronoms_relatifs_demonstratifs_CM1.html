① Les pronoms relatifs — qui, que, dont, où
🔗 Pronoms relatifs

Ils relient deux informations sur le même nom. Ils évitent la répétition.

qui = sujet · que = COD · dont = avec "de" · où = lieu/temps

Le chat qui dort. / Le livre que je lis. / L'ami dont je parle. / La ville où j'habite.

Astuce qui/que : remplace par il/elle → qui · remplace par le/la → que
② Les pronoms démonstratifs — celui, celle, ceux, celles
👉 Pronoms démonstratifs

Ils remplacent un nom déjà mentionné. Ils s'accordent avec ce nom.

celui (m.s.) · celle (f.s.) · ceux (m.p.) · celles (f.p.)

J'ai deux stylos. Celui en rouge est le mien.

Les pronoms : relatifs et démonstratifs

Les pronoms servent à éviter les répétitions. Aujourd'hui tu vas découvrir deux grandes familles : les pronoms relatifs (qui, que, dont, où) qui enrichissent une phrase, et les pronoms démonstratifs (celui, celle…) qui désignent quelque chose de précis !

📚 CM1⏱️ ~20 min ⭐⭐ Intermédiaire🎯 Français — Grammaire

📖 La leçon

① Les pronoms relatifs : qui, que, dont, où

Un pronom relatif sert à relier deux informations sur le même nom, sans le répéter. Il introduit une proposition relative qui donne des détails sur ce nom.

🔗 qui — le sujet

Le chat qui dort est à moi.
→ qui = le chat (il dort). Test : remplace par il ✅

🔗 que — le COD

Le livre que je lis est super.
→ que = le livre (je lis le livre). Test : remplace par le ✅

🔗 dont — avec "de"

L'ami dont je parle est sympa.
→ dont = je parle de l'ami. Il y a de dans la phrase.

🔗 où — lieu ou temps

La ville où j'habite est grande.
→ où = dans la ville. Toujours pour un endroit ou un moment.
🦉 Astuce qui / que :
— Si tu peux remplacer par il / elle → c'est qui (sujet)
— Si tu peux remplacer par le / la → c'est que (COD)
② Les pronoms démonstratifs : celui, celle, ceux, celles

Un pronom démonstratif remplace un nom déjà mentionné pour désigner quelque chose de précis. Il s'accorde en genre et en nombre avec le nom qu'il remplace.

Singulier
Pluriel
Masculin
celui
ceux
Féminin
celle
celles
J'ai deux stylos. Celuim.s. en rouge écrit mieux.
Regarde ces fleurs. Cellesf.p. du jardin sont magnifiques.
J'ai vu les garçons. Ceuxm.p. qui courent sont mes cousins.
💡 Truc pour choisir : Trouve le nom que tu remplaces → regarde son genre (m/f) et son nombre (s/p) → choisis le pronom qui correspond dans le tableau !
③ Comment les distinguer d'un coup d'œil

🔗 Pronom relatif

Vient juste après un nom et introduit une proposition qui le décrit.

Le chien qui aboie… → qui décrit le chien
La fille que tu vois… → que décrit la fille

👉 Pronom démonstratif

Remplace un nom déjà dit et pointe vers quelque chose de précis.

J'ai 2 chats. Celui noir... → remplace le chat
Celles qui chantent... → remplace les filles
🦉

La leçon est terminée ! Pour choisir entre qui et que : remplace par il ou le et vois ce qui marche !
