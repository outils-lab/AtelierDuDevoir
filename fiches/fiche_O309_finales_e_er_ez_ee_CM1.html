Les finales en -é / -er / -ez / -ée

Tous ces mots se prononcent pareil… mais ils ne s'écrivent pas de la même façon ! Apprends à les distinguer.

📚 CM1/CM2 ⏱️ ~25 min ⭐⭐⭐ Difficulté 🎯 Français — Orthographe

📖 La leçon

① -é et -ée : le participe passé

🟣 -é : participe passé masculin singulier

Il est partié. → masculin sing. → -é
Le gâteau est mangé. → masculin sing. → -é

Test : remplace par le participe passé d'un verbe en -ir → fini. Si ça marche → -é.

🔴 -ée : participe passé féminin singulier

Elle est arrivée. → féminin sing. → -ée
La lettre est postée. → féminin sing. → -ée

Test : remplace par finie. Si ça marche → -ée.

🦉 Astuce : Remplace par fini / finie.
→ fini → -é · → finie → -ée
② -er : l'infinitif · -ez : la 2e personne du pluriel

🟡 -er : infinitif du verbe

Il aime chanter. → infinitif → -er
Tu vas manger ce soir. → infinitif → -er

Test : remplace par vendre ou prendre. Si ça marche → -er.

🟢 -ez : 2e personne du pluriel (vous)

Vous chantez très bien ! → vous → -ez
Mangez vos légumes ! → vous (implicite) → -ez

Test : si on peut ajouter "vous" devant → -ez.

🦉 Astuce : Remplace par vendre / vendez.
→ vendre → -er · → vendez → -ez
③ Récap — les 4 tests

🟣 Les 4 tests d'Ugo

-é → remplace par fini → participe passé masc.
-ée → remplace par finie → participe passé fém.
-er → remplace par vendre → infinitif
-ez → remplace par vendez → vous + présent

🟡 Exemples récapitulatifs

Il faut ranger sa chambre. → vendre ✅ → -er
Le lit est rangé. → fini ✅ → -é
La chambre est rangée. → finie ✅ → -ée
Vous rangez vite. → vendez ✅ → -ez
🦉 Méthode en 2 étapes :
1. Est-ce un verbe ? → -er (infinitif) ou -ez (vous)
2. Est-ce un adjectif/participe ? → -é (masc.) ou -ée (fém.)
🔊

Teste chaque mot : fini/finie pour -é/-ée · vendre/vendez pour -er/-ez !

Vous parltrop fort !
