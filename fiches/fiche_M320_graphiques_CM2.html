Lire et construire des graphiques

Diagrammes en barres, en lignes, circulaires… Apprends à lire des données et à choisir le bon graphique !

CM2⏱ 25 min ⭐⭐ DifficultéMathématiques

📖 La leçon

① Le diagramme en barres (histogramme)
🏫 Dans la classe, combien d'élèves préfèrent chaque sport ? On dessine une barre pour chaque sport — plus la barre est haute, plus il y a d'élèves. C'est le diagramme en barres, parfait pour comparer des catégories !

🔑 Lire un diagramme en barres

1. Lire le titre pour comprendre le sujet
2. Lire les axes : axe horizontal = catégories, axe vertical = valeurs
3. Pour chaque barre, lire la hauteur sur l'axe vertical
4. Comparer les barres entre elles pour trouver le maximum, minimum…

Lecture "Foot" a la barre la plus haute → 15 élèves préfèrent le foot. Total = 15+10+6+10+4 = 45 élèves
② Le diagramme linéaire (courbe)
🌡️ La température chaque mois de l'année : on relie des points avec une ligne. Quand ça monte → la valeur augmente. Quand ça descend → elle baisse. Le diagramme linéaire est parfait pour montrer une évolution dans le temps !

🔑 Lire un diagramme linéaire

• La courbe monte → la valeur augmente
• La courbe descend → la valeur diminue
• Le sommet de la courbe = valeur maximale
• Le creux de la courbe = valeur minimale
• Utilisé pour les données qui évoluent dans le temps

💡 Diagramme linéaire vs barres : Barres = comparer des catégories distinctes. Linéaire = montrer une évolution (températures, ventes, croissance…).
③ Lire un tableau de données
📋 Avant de construire un graphique, les données sont souvent dans un tableau. Apprends à extraire les informations importantes : total, maximum, minimum, comparaisons…
Mois Jan Fév Mar Avr Mai Jun
Ventes (unités) 120 95 140 180 165 90

Analyser un tableau

• Maximum : Avr = 180 unités (valeur la plus grande)
• Minimum : Jun = 90 unités (valeur la plus petite)
• Total : 120+95+140+180+165+90 = 790 unités
• Évolution : Jan→Avr = hausse, Avr→Jun = baisse

⚠ Piège de l'échelle ! Sur un graphique, si l'axe ne commence pas à 0, les barres peuvent être trompeuses. Toujours vérifier l'échelle avant de comparer !
