Imparfait vs Passé composé

Apprends à choisir le bon temps : imparfait pour ce qui durait, passé composé pour les événements précis.

📖 La leçon

① Rappel — les deux temps du passé

Tu connais déjà l'imparfait et le passé composé. Le secret, c'est de savoir lequel choisir selon ce qu'on veut exprimer.

⏳ Imparfait — ce qui durait

🔁 Habitude : Chaque soir, il lisait un livre.
🎨 Description : Le ciel était bleu et il faisait chaud.
⏳ Action en cours : Elle dormait quand il est arrivé.

✅ Passé composé — ce qui s'est passé

🎯 Événement ponctuel : Soudain, la porte a claqué.
✅ Action terminée : Hier, nous avons gagné le match.
⚡ Rupture dans un récit : Il lisait quand le téléphone a sonné.
② Le duo imparfait + passé composé dans un récit

Dans un récit au passé, les deux temps travaillent ensemble :

Il faisait nuit. Le vent soufflait fort. Soudain, une lumière est apparue. Tom a regardé par la fenêtre.
Imparfait = décor, état
Passé composé = événements

🦉 Le test : Pose-toi la question — l'action se répétait ou durait (→ imparfait) ? Ou c'est un événement précis qui s'est passé une seule fois (→ passé composé) ?
