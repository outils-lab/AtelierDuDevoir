📖 La leçon

① Qu'est-ce que le passé composé ?

Le passé composé raconte une action passée et terminée. Il se compose toujours de deux éléments : un auxiliaire (avoir ou être) + un participe passé.

🧱 Structure : auxiliaire + participe passé

Elle a mangé une pomme. → avoir + mangé
Il est parti tôt. → être + parti
Nous avons fini nos devoirs. → avoir + fini
② AVOIR ou ÊTRE : comment choisir ?

La majorité des verbes utilisent AVOIR. Mais les verbes de mouvement et de changement d'état utilisent ÊTRE :

allervenirpartirarriver naîtremourirtomberrester entrersortirmonterdescendre
🦉 Moyen mnémotechnique : pense à la maison du Dr. VANDERTRAMP : V(enir) A(ller) N(aître) D(escendre) E(ntrer) R(ester) T(omber) R(etourner) A(rriver) M(ourir) P(artir) → tous avec ÊTRE !
③ Former le participe passé
Verbes en -er-é : chant→ chanté
Verbes en -ir-i : fin→ fini
fairefait
prendrepris
voirvu
venir / naîtrevenu / né
④ L'accord du participe passé avec ÊTRE

📌 Règle : avec ÊTRE → accord avec le sujet

♂
Masculin singulier : il est parti
♀
Féminin singulier : elle est partie (+e)
♂♂
Masculin pluriel : ils sont partis (+s)
♀♀
Féminin pluriel : elles sont parties (+es)
💡 Avec AVOIR : le participe ne s'accorde généralement PAS avec le sujet. Elle a chanté (pas chantée) !
