Les doublets consonantiques

Pourquoi écrit-on balle avec deux l et bal avec un seul ? Apprends les règles du doublement !

📚 CM1 ⏱️ ~20 min ⭐⭐ Difficulté 🎯 Français — Orthographe

📖 La leçon

① Pourquoi une consonne se double-t-elle ?

🔵 La règle du doublement après une voyelle courte

balle → le "a" est court → on double le l
sonner → le "o" est court → on double le n
chatte → le "a" est court → on double le t

En français, une consonne se double souvent après une voyelle courte pour "fermer" la syllabe.

🟡 Les consonnes qui se doublent le plus souvent

llnnttssrrppffmmcc
ballon · annnée · botte · classe · terre · appel · offre · comme · accord
🦉 Astuce : Quand tu entends une voyelle courte avant une consonne, pense à vérifier si cette consonne se double !
② Les familles de mots : un outil précieux

🟢 La famille du mot peut t'aider

honneur → honnorable → déshonnête ✅ même famille, même doublement
sonner → sonnette → résonner ✅
chatte → chat (masculin) → chatton ✅

Si tu connais un mot de la même famille, son orthographe te guide !

🔵 Exceptions : consonnes qui ne se doublent jamais

Le v, le x, le k, le q, le j, le h et le w ne se doublent jamais en français.
voyage · fox · képi · quai · jardin · herbe → jamais de doublement ✅
🦉 Moyen mnémotechnique : Voici Xav qui Kit son Quart de Jeu avec Humour et Wifi → ces 7 lettres ne doublent jamais !
③ Les cas fréquents à mémoriser

🔵 Préfixes qui provoquent un doublement

in- + n → innovation · innocer · innombrable
il- (devant l) → illisible · illégal · illuminer
ir- (devant r) → irréel · irrégulier · irresistible

🟡 Mots courants à bien retenir

additionattentioncarottegommette classiqueterrassenoisetteboisson difficileofficiereffaceraffiche
🦉 Récap :
→ Voyelle courte avant une consonne → penser au doublement
→ Famille du mot = meilleur indice
→ v, x, k, q, j, h, w → jamais de doublement
🔤

Souviens-toi : voyelle courte → pense au doublement ! Et vérifie dans la famille du mot.
