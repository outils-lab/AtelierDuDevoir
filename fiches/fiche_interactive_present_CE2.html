C'est quoi le présent de l'indicatif ?

On utilise le présent pour parler de ce qui se passe maintenant, de ce qui est toujours vrai, ou de ce qu'on fait régulièrement.
→ Je mange une pomme. (maintenant)
→ Le soleil brille chaque matin. (toujours vrai)
→ Nous jouons au foot le mercredi. (habitude)
🟢

Les verbes du 1er groupe (–ER)

Les verbes du 1er groupe se terminent par –ER à l'infinitif.
Exemples : chanter, jouer, manger, aimer, danser…
J'enlève le –ER de l'infinitif et j'ajoute les terminaisons.
🎵 CHANTER
jechante
tuchantes
il / elle / onchante
nouschantons
vouschantez
ils / elleschantent
⚽ JOUER
jejoue
tujoues
il / elle / onjoue
nousjouons
vousjouez
ils / ellesjouent
Les terminaisons du 1er groupe : –e · –es · –e · –ons · –ez · –ent
🔵

Les verbes du 2e groupe (–IR + issons)

Les verbes du 2e groupe se terminent par –IR et font –issons à « nous ».
Exemples : finir, choisir, grandir, rougir, réussir…
✅ FINIR
jefinis
tufinis
il / elle / onfinit
nousfinissons
vousfinissez
ils / ellesfinissent
🌱 GRANDIR
jegrandis
tugrandis
il / elle / ongrandit
nousgrandissons
vousgrandissez
ils / ellesgrandissent
⭐

ÊTRE et AVOIR — à apprendre par cœur !

Ces deux verbes sont très irréguliers. Il n'y a pas de règle — il faut les mémoriser !
⭐ ÊTRE
jesuis
tues
il / elle / onest
noussommes
vousêtes
ils / ellessont
⭐ AVOIR
j'ai
tuas
il / elle / ona
nousavons
vousavez
ils / ellesont

Verbes irréguliers importants

Ces verbes ne suivent pas les règles — ils doivent être mémorisés !
🚶 ALLER
jevais
tuvas

ils/ellesvont
✏️ FAIRE

tufais

ils/ellesfont
🏃 VENIR
jeviens
tuviens
il/ellevient
nousvenons
vousvenez
ils/ellesviennent
Pour savoir si un verbe est du 2e groupe : est-ce que « nous » fait –issons ? Si oui → 2e groupe. Sinon → irrégulier (3e groupe).
