① Poser une soustraction
🦉

Pour soustraire 77 − 45, on pose comme pour l'addition : le grand nombre en haut, le petit en bas. Unités sous unités, dizaines sous dizaines ! 📐

2
Je soustrais les unités d'abord : 7 − 5 = 2. J'écris 2 sous les unités.
3
Je soustrais les dizaines : 7 − 4 = 3. J'écris 3 sous les dizaines.

Résultat : 32. Vérif : 32 + 45 = 77 ✓
La règle d'or :
🔴 Le grand nombre EN HAUT !
On commence par les unités (à droite).
Sans retenue : chaque chiffre du bas est plus petit que celui du haut.

③ Vérifier avec l'addition !
🦉

Super astuce : pour vérifier une soustraction, tu fais l'addition à l'envers ! Si 77 − 45 = 32, alors 32 + 45 doit faire 77. 🔄

SOUSTRACTION
77 − 45 = 32
🔄
VÉRIFICATION
32 + 45 = 77 ✓
Pour vérifier :
résultat + nombre du bas = nombre du haut
Ex : 32 + 45 = 77 ✓ → la soustraction est juste !
