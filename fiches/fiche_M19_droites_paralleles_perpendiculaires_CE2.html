📖 La leçon

① Les droites parallèles
🦉

Deux droites sont parallèles quand elles vont dans la même direction et ne se croisent jamais, même si on les prolonge ! 😊

✅ Parallèles
horizontales
✅ Parallèles
verticales
Exemples dans la vie : les rails d'un train, les lignes d'un cahier, les bords d'une route → toujours parallèles !
Symbole : on dessine de petites flèches identiques sur les droites parallèles.
② Les droites perpendiculaires
🦉

Deux droites sont perpendiculaires quand elles se croisent en formant un angle droit (90°). On vérifie avec l'équerre ! 📐

✅ Perpendiculaires
angle droit ✓
❌ Pas perpendiculaires
pas d'angle droit
Exemples : le bord d'une feuille, les coins d'un carré, le signe +, la croix rouge → toujours perpendiculaires !
Symbole : on dessine un petit carré à l'angle droit.
③ Résumé et exemples du quotidien
🦉

Voici comment les différencier d'un coup d'œil ! 👀

Parallèles : rails de train 🚂, lignes d'un cahier, les bords d'une rue
Perpendiculaires : coins d'une feuille 📄, croix ✚, signe +, bords d'un carré
Astuce : tu peux vérifier avec une équerre !
→ Pose l'équerre entre les deux droites
→ Si l'angle coïncide parfaitement → perpendiculaires
→ Si les droites ne se croisent jamais → parallèles
