L'accord du participe passé

Avec être ou avoir ? Le COD est avant ou après ? Apprends à accorder correctement !

📚 CM2 / 6e ⏱️ ~25 min ⭐⭐⭐ Difficulté 🎯 Français — Conjugaison

📖 La leçon

① La règle avec l'auxiliaire ÊTRE

🔵 Avec ÊTRE : accord avec le sujet

Il est parti. → Elle est partie. → Ils sont partis. → Elles sont parties.
Nous sommes arrivés. (masculin pluriel)
Vous êtes rentrées. (féminin pluriel)
🦉 Astuce Ugo : Avec être → pose-toi la question : qui est-ce qui est parti(e)(s) ? C'est le sujet → on accorde avec lui !
② La règle avec l'auxiliaire AVOIR

🟡 COD après le verbe → PAS d'accord

Elle a mangé une pomme. (quoi ? → une pomme, après)
Ils ont acheté des livres. (COD après → invariable)

🟢 COD avant le verbe → ACCORD

La pomme qu'elle a mangée. (que = la pomme, féminin, avant)
Les livres qu'ils ont achetés. (que = les livres, masculin pluriel, avant)
💡 Avec avoir : cherche si un COD est placé AVANT le verbe. Si oui → accord avec ce COD. Si non → pas d'accord.
③ Les verbes qui se conjuguent avec ÊTRE

À apprendre : les 14 verbes de déplacement et d'état

aller / venirallé · venu
partir / arriverparti · arrivé
entrer / sortirentré · sorti
naître / mourirné · mort
monter / descendremonté · descendu
rester / tomberresté · tombé
retourner / passerretourné · passé
+ tous les pronominauxse lever → s'est levé(e)
