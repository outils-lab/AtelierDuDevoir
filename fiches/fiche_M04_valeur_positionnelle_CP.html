La place des chiffres !

Le même chiffre peut valoir 3 ou 30 selon sa place. C'est ça la valeur positionnelle ! 🌟

CP / CE1 ⭐ Facile 🔢 Maths

📖 La leçon

① La place change la valeur !
🦉

Regarde ce chiffre : 3. Selon sa place, il vaut 3 ou 30 ! C'est très important de regarder où il est dans le nombre. 😊

35
Le 3 est à gauche
→ il vaut 30
53
Le 3 est à droite
→ il vaut 3
La règle :
Le même chiffre a une valeur différente selon sa place !
Le 3 dans 35 → vaut 30 (dizaine)
Le 3 dans 53 → vaut 3 (unité)
② Quelle valeur ?
🦉

Pour trouver la valeur d'un chiffre, regarde sa colonne dans le tableau ! 📊

47
Dizaines
4
Unités
7
47 →
Le 4 est dans les dizaines → vaut 40
Le 7 est dans les unités → vaut 7
62 →
Le 6 est dans les dizaines → vaut 60
Le 2 est dans les unités → vaut 2
90 →
Le 9 → vaut 90 (dizaines)
Le 0 → vaut 0 (unités, rien !)
À retenir :
Chiffre dans les dizaines → valeur = chiffre × 10
Chiffre dans les unités → valeur = chiffre tout seul
Le 0 vaut toujours 0, quelle que soit sa place !
③ Décomposer un nombre
🦉

Décomposer, c'est séparer la valeur des dizaines et la valeur des unités. Comme défaire un sandwich en deux ! 🥪

56 = 50 + 6 (5 diz. + 6 unit.)
73 = 70 + 3 (7 diz. + 3 unit.)
40 = 40 + 0 (4 diz. + 0 unit.)
Astuce : 56 = 50 + 6. On retrouve 50 en faisant 5 × 10. C'est parce que le 5 est dans la colonne des dizaines (× 10) !
