📖 La leçon

① Le futur simple — à quoi ça sert ?

Le futur simple exprime ce qui se passera plus tard — une action future certaine, une promesse ou une prévision.

🔮 Demain, nous partirons en vacances.
🔮 Dans dix ans, elle sera médecin.
🔮 Tu mangeras tes légumes si tu veux du dessert !
demainbientôtdans X ans l'année prochaineun jourquand tu seras grand
② Comment former le futur simple

🍳 La recette pour les verbes réguliers

1
Prends l'infinitif complet du verbe (chanter, finir…)
2
Ajoute la terminaison selon le pronom :
je-ai → chanterai
tu-as → chanteras
il / elle-a → chantera
nous-ons → chanterons
vous-ez → chanterez
ils / elles-ont → chanteront
💡 Verbes en -re : on enlève le -e final avant d'ajouter. prendre → prendr + ai = je prendrai
③ Les radicaux irréguliers à connaître !

Ces verbes n'utilisent pas leur infinitif comme radical — il faut les apprendre par cœur :

êtreser- → je serai
avoiraur- → j'aurai
allerir- → j'irai
fairefer- → je ferai
voirverr- → je verrai
venirviendr- → je viendrai
pouvoirpourr- → je pourrai
vouloirvoudr- → je voudrai
🦉 Astuce : Les terminaisons (-ai, -as, -a, -ons, -ez, -ont) sont toujours les mêmes — même pour les irréguliers. Seul le radical change !
